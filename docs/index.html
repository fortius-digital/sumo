SUMO Connected Outcomes

The Simple Unified Model for Orthopaedics is a framework for organising data so that they maintain their complex relationships with other data without making the system complex. Traditonal systems strip and flatten information as they import and save, wrestling data into a two dimensional table format. In an effort to be more efficient they lose the context of data, believing it can be recreated with complex queries.

SUMO treats data and their relationships with equal importance, creating a densely connected graph of information - like nodes in a network. SUMO couples a flexible and robust data model with highly efficient algorithms traverse the graph.
